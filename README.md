测试助手
===================
一站式网页测试工具，使用Django管理及录制测试脚本，同时，使用录制的测试脚本测试用例。

Author: Bruce He <bruce@shbewell.com>

Version: `0.1`

Requirements
-------------
* Python(3.4,3.5,3.6)
* selenium(3.13.0)
* Django(2.0.6)

Installation
------------

需要安装selenium,Django,django-admin-sortable2

```bash
pip install selenium
pip install Django
pip install django-admin-sortable2

```
